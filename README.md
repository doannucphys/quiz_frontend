## Quiz frontend code 

### Code

vuejs (vue3)

### Install library
```
npm i
```

### Run on dev environment
```
npm run dev
```
code will run on port 5173 (http://localhost:5173)

### Build code 
```
npm run build 
```

### Run built code
```
npm run preview
```
code will run on port 4173 (http://localhost:4173)
